# Integration guide for Windows Agent


<Note>
**Important for Windows Server 2016 Users**  
If you are using **Windows Server 2016**, `curl` is **not installed by default**.  
Follow the steps in the [cURL Installation](../integrations/config_curl) section of this guide.
</Note>

## Installing Windows agent

### 1. Check pre-installation requirements

    * Compatible with **Windows Server version 2016** or higher
    * **The UTMStack agents communicate over ports 9000 and 50051**. Please make sure these ports are open.
