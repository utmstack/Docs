# Licensing FAQ

Welcome to the Licensing FAQ for UTMStack version 10.9.0. This document provides answers to frequently asked questions regarding licensing and subscription for the UTMStack platform. For additional support, please visit our [support page](https://support.utmstack.com).

## Frequently Asked Questions

### What is UTMStack?

UTMStack is an open-source security platform designed for Ubuntu 22.04 LTS. It utilizes encrypted communications, containerized microservices, and strong authentication to ensure secure operations. The platform supports scalable resource recommendations based on log storage needs.

### What is the licensing model for UTMStack?

UTMStack is licensed under the AGPL version 3. For more detailed information, please refer to the [LICENSE](LICENSE) file.

### What are the differences between the Open Source and Enterprise versions of UTMStack?

The Enterprise version of UTMStack includes additional features that are beneficial for enterprises and managed service providers (MSPs). These features are tailored to enhance the platform's capabilities beyond the open-source offering.

### Does UTMStack use ELK for log correlation?

No, UTMStack does not use ELK for log correlation. Instead, it features a proprietary correlation engine built from scratch to analyze data before ingestion, maximizing real-time correlation capabilities.

### Is UTMStack based on Grafana, Kibana, or a similar reporting tool?

UTMStack is not based on Grafana, Kibana, or similar tools. It has been developed from the ground up as a simple and intuitive Security Information and Event Management (SIEM) and Extended Detection and Response (XDR) platform.
