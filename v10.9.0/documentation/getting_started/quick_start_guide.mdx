# Quick Start Guide for UTMStack v10.9.0

Welcome to the Quick Start Guide for UTMStack version 10.9.0. This guide will help you get up and running with UTMStack in approximately 15 minutes by providing you with essential setup instructions and resources.

## Introduction

UTMStack is an open-source security platform designed for Ubuntu 22.04 LTS. It utilizes encrypted communications, containerized microservices, strong authentication, and multiple credential protections. This guide is part of the comprehensive UTMStack documentation, which includes system requirements, setup instructions, and integration options.

For more detailed guides and resources, visit the [UTMStack Documentation](https://documentation.utmstack.com/v10/Getting%20started/introduction).

## System Requirements

Before installing UTMStack, ensure that your system meets the necessary hardware and software prerequisites. The system requirements documentation provides detailed information on configuration prerequisites for effective installation and operation.

For more details, refer to the [System Requirements Documentation](https://documentation.utmstack.com/v10/Installation/system_requirements).

## Installation Steps

UTMStack can be installed using an installer file or an ISO image. The following instructions are for the installer file option:

1. **Download the Installer**: Obtain the UTMStack installer file from the official website.
2. **Run the Installer**: Execute the installer file on your Ubuntu 22.04 LTS system.
3. **Follow the Prompts**: Complete the installation by following the on-screen instructions provided by the installer.

For installation using an ISO image, please refer to the [UTMStack Installation Guide](https://utmstack.com/install).

## Resource Recommendations

UTMStack's resource recommendations are based on log storage needs. Here are some guidelines:

- **50 Data Sources**: 4 Cores, 16 GB RAM, 150 GB Disk Space
- **120 Data Sources**: 8 Cores, 16 GB RAM, 250 GB Disk Space
- **240 Data Sources**: 16 Cores, 32 GB RAM, 500 GB Disk Space
- **500 Data Sources**: 32 Cores, 64 GB RAM, 1000 GB Disk Space

For setups exceeding 500 data sources, consider adding secondary nodes for horizontal scaling.
