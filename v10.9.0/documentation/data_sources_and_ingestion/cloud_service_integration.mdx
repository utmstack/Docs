# Cloud Service Integration

## Introduction

This guide provides instructions for integrating cloud service logs from AWS, Azure, GCP, and other platforms into UTMStack. UTMStack is an all-in-one cybersecurity platform designed for hybrid environments, offering integrated services such as SIEM, threat detection, compliance management, and cloud monitoring. It supports flexible, no-code customizable dashboards and reports for security monitoring, compliance, forensic analysis, and uptime management.

## Supported Integrations

UTMStack supports broad integration with various platforms and systems, enabling comprehensive monitoring and analysis without the need for custom coding or complex configurations. The supported integrations include:

- **Cloud Platforms**: Azure, AWS
- **Virtualization**: Hypervisors like KVM, HyperV, VMWare
- **Physical Infrastructure**: Datacenter components
- **Software**: SharePoint, SQL Server
- **Operating Systems**: Windows and Linux servers and endpoints
- **Applications**: PaaS and SaaS applications like Office365
- **Proprietary Devices**: CISCO, Sophos
- **Container Orchestration**: Kubernetes, Docker

## Integration Configuration

### API Endpoints for UTM Integrations

UTMStack provides API endpoints for managing integrations, allowing users to retrieve, create, and update UTM integration records. The API supports query parameters for filtering and pagination.

- **Retrieve All Integrations**: Use the `GET /api/utm-integrations` endpoint with query parameters for criteria and pagination.
- **Create a New Integration**: Use the `POST /api/utm-integrations` endpoint with the required JSON schema.
- **Update an Existing Integration**: Use the `PUT /api/utm-integrations` endpoint with the required JSON schema.

## Compliance and Reporting

UTMStack streamlines compliance management by offering built-in reports and dashboards for major standards such as HIPAA, GDPR, and GLBA. Users can monitor security alerts, account activity, and generate necessary reports for auditors. The compliance management features include:

- **HIPAA**: Security management process, audit controls, and relevant security alerts.
- **GDPR**: Logon activity, vulnerability reports, and data privacy monitoring.
- **GLBA**: Successful login/logoff tracking, file access, and policy changes.

## Advanced Data Visualization and Reporting

UTMStack's proprietary data visualization and analysis engine allows users to create, modify, and delete dashboards and reports without writing code. This flexibility supports:

- **Custom Dashboards**: Build or modify dashboards to fit unique organizational needs.
- **Real-time Data Exploration**: Analyze systems data in near real-time and respond to incidents.
- **Forensic Analysis**: Use log and event explorers for detailed forensic investigations.
