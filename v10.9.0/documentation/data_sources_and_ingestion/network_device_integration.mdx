# Network Device Integration

## Introduction

UTMStack is a comprehensive cybersecurity platform designed to provide integrated security services across hybrid environments. This guide focuses on integrating network devices such as firewalls, routers, and switches into UTMStack, enhancing security monitoring and management capabilities.

## Features of UTMStack

UTMStack offers a wide array of features that are essential for robust network security and management:

- **Log Management (SIEM):** Real-time log collection and correlation, along with a powerful dashboard and report builder for personalized monitoring and compliance auditing.
- **Threat Detection and Response:** Utilizes rule-based and heuristic analysis for network and host intrusion detection, supported by machine learning algorithms for anomaly detection.
- **Compliance Management:** Provides built-in reports and dashboards for compliance with standards like HIPAA, GLBA, SOC 2, GPDR, FISMA, CMMC, and PCI-DSS.
- **Threat Intelligence:** Correlates threat intelligence data to identify and mitigate risks from spam, malware, botnets, and other cyber threats.

## Supported Integrations

UTMStack supports integration with a variety of network devices and platforms, enabling centralized management and enhanced security monitoring. Supported devices include:

- **Firewalls and Security Appliances:** CISCO, FORTIGATE, SOPHOS_XG, SONIC_WALL, PALO_ALTO.
- **Routers and Switches:** CISCO_SWITCH, MIKROTIK.
- **Endpoint Security:** ESET, KASPERSKY, BITDEFENDER.
- **Cloud and SaaS Monitoring:** Azure, AWS, Office365, Google Cloud.

## Integration Process

To integrate network devices with UTMStack, follow these general steps:

1. **Access the UTMStack Interface:** Log in to your UTMStack instance using the credentials provided during the installation process.
2. **Navigate to the Integrations Menu:** Go to the Integrations menu and click on "+ Enable Integration."
3. **Configure Device Integration:** Select the network device you wish to integrate and follow the on-screen instructions to complete the setup.
4. **Verify Configuration:** Ensure that the integration is correctly configured by checking the module configuration and monitoring the data flow from the device to UTMStack.

## Advanced Data Visualization and Reporting

UTMStack provides flexible, no-code customizable dashboards and reports for security monitoring, compliance, forensic analysis, and uptime management. Users can create, modify, and delete dashboards and reports without writing a single line of code, leveraging a proprietary data visualization and analysis engine.

## System Requirements

To ensure optimal performance, UTMStack requires specific hardware configurations based on data volume:

- **Data Sources:** UTMStack can handle approximately 50 data sources generating 100 GB of data per month.
- **Server Specifications:** Recommended hardware includes 8 cores, 16 GB RAM, and 150 GB disk space for managing 50 data sources.

Integrating network devices into UTMStack enhances your organization's security posture by providing comprehensive monitoring, threat detection, and compliance management capabilities.
