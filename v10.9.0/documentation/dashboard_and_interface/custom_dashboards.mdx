# Custom Dashboards in UTMStack

## Overview

The UTMStack Dashboard is a centralized interface designed to monitor user activity, alerts, vulnerabilities, and network traffic. It allows users to create, customize, and analyze security visualizations, providing critical insights for effective network protection. This guide will walk you through the process of creating and customizing dashboards to suit your specific needs.

## Getting Started

To begin using the UTMStack Dashboard:

1. Log into your UTMStack account.
2. Navigate to the **Dashboard** tab.

![Dashboard Menu](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardmenu.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=67a9429d6db0f2bb381cafaa4d43b9b1)

## Creating a New Dashboard

### Step 1: Initiate Dashboard Creation

- Click on **New Dashboard** to open a pop-up window displaying a list of pre-created visualizations.

![New Dashboard](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardnewvisualization.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=e65f182ea870e94474f70ddad0ab0e61)

### Step 2: Select Visualizations

- Choose from the list of pre-made visualizations by checking the respective checkboxes.
- Example visualizations include **Top 5 User Logon Failures** and **Detail User Logon Failure**.

![Visualizations List](https://mintcdn.com/utmstack/lxb2uSs0wybYno8q/images/dashboards/visualizationslist.png?fit=max&auto=format&n=lxb2uSs0wybYno8q&q=85&s=641be0acdd5c2c14c95582fe627a1f6d)

### Step 3: Customize and Save

- After selecting your visualizations, click the **+ Add visualizations** button.
- Customize the dashboard by modifying the size, position, and date range of visualizations.
- Once satisfied, click the **Save Dashboard** button, providing a name and optional description.

![Dashboard Editor View](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboareditorview.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=fdd96790ae3104a8a286562ce2016bdd)

## Managing Dashboards

### Dashboard List

- Use the **Dashboard List** to search, view, and manage your dashboards.
- The list includes functionalities such as search, copy, edit, and delete.

![Dashboard List](https://mintcdn.com/utmstack/lxb2uSs0wybYno8q/images/dashboards/mydashboard.png?fit=max&auto=format&n=lxb2uSs0wybYno8q&q=85&s=3aba90bc025dba4312583eb6c6212fb1)

### Exporting Dashboards

- Dashboards can be exported to PDF using the **Save to PDF** button for easy sharing and reporting.

![Dashboard View](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardview.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=ca648941061cdddc115dd2615de60882)

## Importing Dashboards

- Use the **Import Dashboard** function to reuse exported dashboards across different sections or instances, facilitating easy duplication and setup.

![Import Dashboard](https://mintcdn.com/utmstack/lxb2uSs0wybYno8q/images/dashboards/importdash.png?fit=max&auto=format&n=lxb2uSs0wybYno8q&q=85&s=3866286a1d58b2480a13c31f72db199b)

## Summary

The UTMStack Dashboard is a powerful tool for creating custom security visualizations, allowing for detailed monitoring and analysis of network activity. By following the steps outlined in this guide, users can efficiently create, customize, and manage dashboards to enhance their security operations.

For more detailed information, visit the [UTMStack Dashboard Documentation](https://documentation.utmstack.com/v10/dashboards/dashboard).
