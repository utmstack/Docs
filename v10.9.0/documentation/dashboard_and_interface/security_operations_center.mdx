# Security Operations Center Dashboard Guide

## Introduction

Welcome to the UTMStack Security Operations Center (SOC) Dashboard Guide. This guide is designed to help you navigate and utilize the UTMStack user interface for effective threat monitoring. UTMStack is a comprehensive open-source Unified Threat Management platform that integrates essential security services such as SIEM, threat detection, incident response, compliance management, and advanced reporting to enhance cybersecurity.

## Features of UTMStack

UTMStack offers a wide range of features to support security operations:

- **Log Management (SIEM):** Real-time log collection and correlation.
- **Threat Detection and Response:** Advanced detection capabilities using rule-based and heuristic analysis engines.
- **Compliance Management:** Built-in reports and dashboards for compliance with standards like HIPAA, GLBA, SOC 2, and more.
- **Incident Response and Endpoint Protection:** Automated and on-demand incident response capabilities.
- **Network and Host Intrusion Detection:** Rule-based and heuristic analysis for network and host systems.
- **Threat Intelligence:** Correlation with threat intelligence databases to identify and mitigate threats.
- **Advanced Data Visualization and Reporting:** Customizable dashboards and reports for security monitoring and analysis.

## Using the SOC Dashboard

The UTMStack Dashboard provides a centralized interface for monitoring user activity, alerts, vulnerabilities, and network traffic. It allows users to create, customize, and analyze security visualizations for effective network protection.

### Getting Started

1. **Accessing the Dashboard:**
   - After logging into UTMStack, navigate to the **Dashboard** tab.

   ![Dashboard Menu](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardmenu.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=67a9429d6db0f2bb381cafaa4d43b9b1)

2. **Creating a New Dashboard:**
   - Click on **New Dashboard** to start creating custom visualizations.

   ![Dashboard View](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardnewvisualization.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=e65f182ea870e94474f70ddad0ab0e61)

### Customizing Dashboards

UTMStack dashboards are flexible and customizable without the need for coding. Users can create, modify, and delete dashboards and reports to fit their specific security monitoring needs. This flexibility is built on a proprietary data visualization and analysis engine.

### Threat Activity and Intelligence

The dashboard includes features for threat activity monitoring and intelligence investigation:

- **Threat Activity:** Provides insights into ongoing threats and alerts.
- **Threat Intelligence Investigator:** Analyzes threat intelligence data to identify potential security risks.
