# Compliance Frameworks Overview

Welcome to the Compliance Frameworks documentation for version v10.9.0. This page provides an overview of the compliance management capabilities within the UTMStack platform, focusing on support for various industry standards such as PCI DSS, HIPAA, SOX, and others. This documentation is part of the Security and Compliance group, highlighting the platform's security features and compliance requirements.

## Introduction to Compliance Management

The Compliance Management module in UTMStack is designed to help organizations monitor and maintain adherence to multiple industry regulations. It categorizes compliance data and provides dedicated sections for each standard, offering a comprehensive view of an organization’s regulatory compliance status.

## Supported Compliance Standards

UTMStack supports a wide range of compliance standards, each with its own dedicated section and reporting capabilities:

1. **Health Insurance Portability and Accountability Act (HIPAA)**
   - Includes reports on security management processes, audit controls, account validations, and relevant security alerts.

2. **General Data Protection Regulation (GDPR)**
   - Focuses on logon activity, vulnerability reports, potential data exposure, and data privacy compromises.

3. **Gramm-Leach-Bliley Act (GLBA)**
   - Provides reports on successful logins, file access, policy changes, and new user accounts in Active Directory.

4. **System and Organization Controls 2 (SOC 2)**
   - Offers reports aligned with SOC 2 Control Criteria focusing on security, availability, processing integrity, confidentiality, and privacy.

5. **Federal Information Security Management Act (FISMA)**
   - Provides pre-defined reports monitoring compliance with FISMA’s crucial sections.

6. **Cybersecurity Maturity Model Certification (CMMC)**
   - Includes reports specifically designed to monitor compliance with different CMMC Levels.

7. **Payment Card Industry Data Security Standard (PCI-DSS)**
   - Ensures credit card data processing activities comply with PCI-DSS standards.

## Compliance Management Features

- **Customizable Dashboards and Reports**: UTMStack offers flexible, no-code customizable dashboards and reports for security monitoring and compliance.
- **Built-in Reports and Dashboards**: The platform provides several built-in reports and interactive dashboards, reinforced by an event and logs explorer for advanced analysis.
- **Integration Support**: UTMStack supports integration with various platforms and systems, including Azure, AWS, Office365, and more.

## Exporting and Importing Compliance Data

The module allows users to export their compliance configuration to ensure data integrity and import compliance data from other systems using JSON files for seamless integration.
