# Building Your First Dashboard

## Introduction

This tutorial will guide you through the process of creating a custom security dashboard using UTMStack version 10.9.0. The UTMStack Dashboard provides a centralized interface for monitoring user activity, alerts, vulnerabilities, and network traffic. By following this guide, you will learn how to create, customize, and analyze security visualizations for effective network protection.

## Overview

The UTMStack Dashboard is a comprehensive interface that displays a collection of visualizations and searches. It offers critical insights into user activity, alerts, vulnerabilities, and network traffic within your organization. Equipped with a variety of visualization types, UTMStack streamlines security monitoring and analysis for a more secure network environment.

## Getting Started

1. **Log into UTMStack**: Begin by logging into your UTMStack account.
2. **Navigate to the Dashboard Tab**: Once logged in, navigate to the **Dashboard** tab.

   ![Dashboard Menu](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardmenu.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=67a9429d6db0f2bb381cafaa4d43b9b1)

3. **Create a New Dashboard**: Click on the **New Dashboard** button to start creating your custom dashboard.

   ![Dashboard View](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardnewvisualization.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=e65f182ea870e94474f70ddad0ab0e61)

## Creating a New Dashboard

Upon clicking the New Dashboard option, a pop-up window appears on the right side of your screen. This window displays a list of pre-created visualizations tailored to common user needs.

![Visualizations List](https://mintcdn.com/utmstack/lxb2uSs0wybYno8q/images/dashboards/visualizationslist.png?fit=max&auto=format&n=lxb2uSs0wybYno8q&q=85&s=641be0acdd5c2c14c95582fe627a1f6d)

1. **Select Visualizations**: Choose your preferred visualizations by checking the respective checkboxes. For example, you might select visualizations like **Top 5 User Logon Failures** and **Detail User Logon Failure**.
2. **Add Visualizations**: Click the **+ Add visualizations** button at the bottom of the window.

   ![Save Visualizations](https://mintcdn.com/utmstack/lxb2uSs0wybYno8q/images/dashboards/savevisualizations.png?fit=max&auto=format&n=lxb2uSs0wybYno8q&q=85&s=6fa879406cb65141dbd9d833fb220417)

3. **Customize the Dashboard**: After adding visualizations, you will be directed to the Dashboard view editor. Here, you can modify the size, position, and date range of your visualizations or customize them further to meet your specific requirements.

   ![Dashboard Editor View](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboareditorview.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=fdd96790ae3104a8a286562ce2016bdd)

4. **Save the Dashboard**: Once you have configured your dashboard, click the **Save Dashboard** button. Provide a name for your dashboard and optionally add a description.

## Managing Your Dashboard

- **Locate Your Dashboard**: Use the search bar in the **Dashboard List** to quickly find your newly created dashboard.

  ![My Dashboard](https://mintcdn.com/utmstack/lxb2uSs0wybYno8q/images/dashboards/mydashboard.png?fit=max&auto=format&n=lxb2uSs0wybYno8q&q=85&s=3aba90bc025dba4312583eb6c6212fb1)

- **Interact with Dashboard Elements**: Click on the elements within your dashboard to view the data visualized in the charts.
- **Export to PDF**: Use the **Save to PDF** button to export your dashboard visualizations to a PDF file.

  ![Dashboard View](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/dashboards/dashboardview.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=ca648941061cdddc115dd2615de60882)

## Summary

In this tutorial, you learned how to create a custom security dashboard in UTMStack by selecting and customizing visualizations. You can now efficiently monitor and analyze security data within your organization. For further customization, consider exploring additional visualization options and dashboard settings.

For more detailed information, visit the [UTMStack Dashboard Documentation](https://documentation.utmstack.com/v10/dashboards/dashboard).
